# Mobile Testing Mercurio

Este proyecto corresponde a la Actividad 3 del curso de Mobile Testing.

## Descripción
La aplicación consume una API externa que contiene un listado de productos y muestra su nombre y precio.

## API Consumida
https://jsonkeeper.com/b/MX0A

## Tecnologías utilizadas
- Kotlin
- Gson (para parsear JSON)

## Ejecución
Ejecutar la función `main()` en `main.kt`.
Se mostrará en consola algo similar a:

Producto: iPhone 13 - Precio: 999.99 USD  
Producto: Samsung Galaxy S21 - Precio: 899.99 USD  

Producto: Google Pixel 6 - Precio: 799.99 USD (sin stock)
